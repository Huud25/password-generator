# Password Generator (CLI)

A secure command-line tool that generates strong passwords using Python's `secrets` module.

---

## Why `secrets`?

This project uses `secrets` instead of `random` because `secrets` is designed for security-sensitive data (like passwords and tokens).

---

## Features

- Generates strong passwords with configurable length
- Uses a cryptographically secure generator (`secrets`)
- Simple CLI workflow

---

## How to run

### Requirements
- Python 3 installed

### Steps

```bash
python main.py
```
